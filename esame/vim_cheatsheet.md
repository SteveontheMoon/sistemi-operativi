# VIM
Attenzione: non contiene tutti i comandi necessari per poterlo usare ma solo i comandi che **io** non mi ricordo!
In ogni caso puo' essere utile per gli altri.

## Move to <n> line
| keyboard | command |
| -------- | ------- |
| `nG`	   | exe n	 |

## Go to end of file
`G`

## Go to begin of file
`gg`

## Go to end of line
Insert mode: `A`
Command mode: `$`

## Go to begin of lien
Insert mode: `I`
Comment mode: `^`
