# VIM

## .vimrc
set number
syntax on
set tabstop=4
set colorcolumn=120


## Move to <n> line
| keyboard | command |
| -------- | ------- |
| `nG`	   | exe n	 |

## Go to end of file
`G`

## Go to begin of file
`gg`

## Go to end of line
Insert mode: `A`
Command mode: `$`

## Go to begin of lien
Insert mode: `I`
Comment mode: `^`
